## Download
### `MacOS`
Get latest `release` from the right side of github page.  
And remove quarantine from file with  
`sudo xattr -rd com.apple.quarantine MyRacingGame-MacOS.app`

### `Linux & Windows`
currently officially not supported

## Updates
Game automaticly checks for new versions, after user confirmation download begins with restart and deletion od old files.
