# 2D Racing Game
Simple 2D top down racing game with timed runs and global leaderboard.

## Download
### **MacOS**
1. Download the latest `.zip` from the [Releases](https://github.com/Kdrakula/2D-Racing-Game/releases) page.

`![GitHub release (latest by date)](https://img.shields.io/github/v/release/Kdrakula/2D-Racing-Game)`
2. Open **Terminal** and run the following command to bypass Apple quarantine:
   ```bash
   sudo xattr -rd com.apple.quarantine MyRacingGame-MacOS.app
   ```

### `Linux & Windows`
Status: Currently not officially supported.

## Features
- Timed Runs: Tracking for competitive play.
- Auto-Updates: The game checks for the latest release on startup.
- Global Leaderboard: Compare your best laps with players around the world.

Notes: Global leaderboard and auto-update requires an active internet connection.
